# 참고 자료 인덱스

이 파일은 유용한 참고 자료와 해당 자료를 왜 보는지 정리하는 곳입니다.

## Unreal 공식 참고 자료

- Recommended Asset Naming Conventions
  - Content Browser 에셋 이름과 prefix 규칙을 정할 때 사용합니다.
  - https://dev.epicgames.com/documentation/pl-pl/unreal-engine/recommended-asset-naming-conventions-in-unreal-engine-projects

- Lyra Sample Game
  - 모듈형 게임플레이 아키텍처, Enhanced Input, Gameplay Ability System, UI 레이어링, 확장 가능한 프로젝트 구조를 참고합니다.
  - https://dev.epicgames.com/documentation/unreal-engine/lyra-sample-game-in-unreal-engine

- Gameplay Ability System
  - 추후 능력, Attribute, Gameplay Effect, 쿨다운, 레벨업 능력 구조를 설계할 때 참고합니다.
  - https://dev.epicgames.com/documentation/unreal-engine/gameplay-ability-system

- Primary Asset Labels / Asset Manager
  - 추후 스테이지, 능력, 적 에셋에 명시적 로딩과 쿠킹 규칙이 필요해질 때 참고합니다.
  - https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Runtime/Engine/Engine/UAssetManager/ApplyPrimaryAssetLabels

## 샘플 프로젝트 참고 방향

- Lyra: 아키텍처, Ability, Input 패턴.
- Cropout: 1인 개발 친화적인 범위와 마을/환경 구성.
- Parrot: 2.5D 횡스크롤 이동, 카메라, 전투 참고.
