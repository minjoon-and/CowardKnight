# 게임 기획 요약

## 핵심 판타지

플레이어는 겁쟁이 기사 볼트를 조작합니다.

볼트는 정면으로 용감하게 싸우는 영웅이 아니라, 도망치고 피하고 실수하면서도 결과적으로 적을 물리치는 캐릭터입니다.

핵심 감정 축은 다음과 같습니다.

- 코미디: 볼트는 겁먹었지만 결과는 영웅적입니다.
- 조작감: 플레이어는 직접 이동, 점프, 공격, 능력 선택을 합니다.
- 성장감: 볼트는 약하게 시작하지만 로그라이크 업그레이드를 통해 점점 강해집니다.
- 애착: 마을과 NPC는 플레이어가 지키고 싶어지는 대상이어야 합니다.

## 장르

2.5D 횡스크롤 로그라이크 액션.

첫 playable prototype은 아래 요소에 집중합니다.

- 횡스크롤 이동.
- 점프.
- 기본 근접 공격.
- 적 1종.
- 화이트박스 스테이지 1구역.
- 경험치 획득과 레벨업 선택.

## MVP 방향

MVP는 모든 콘텐츠를 완성하는 빌드가 아닙니다.

핵심 루프가 재미있는지 증명하는 빌드입니다.

핵심 루프:

1. 스테이지 시작.
2. 횡스크롤 레인에서 좌우 이동.
3. 적 웨이브 조우.
4. 적 처치.
5. 경험치 획득.
6. 레벨업.
7. 세 장의 능력 카드 중 하나 선택.
8. 구역 끝에 도달하거나 보스 처치.

## 능력 카테고리

능력 카테고리는 볼트의 겁쟁이 성격을 유지해야 합니다.

- 도주: 이동 속도, 대시, 연막, 패닉 질주.
- 반격: 후퇴하거나 뒤돌아볼 때 발동하는 공격.
- 생존: 보호막, 음식, 운 좋은 회피, 피해 감소.
- 마을 지원: NPC 도움, 함정, 응원, 회복.
